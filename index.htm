Emoji provide a visual reprersentation of the MAGIC bytes PRONOM uses to identify files. Specific bytes and gaps between those become more apparent.

Data is generated from PRONOM and the skeleton-test-suite . More information is available in the GitHub README.
radare2 is a powerful command-line hex-editor and binary explorer that the #digipres community might be interested in exploring further.


To create in radare: r2 -q -c 'pxe {f_sz_bytes}' {filename}
